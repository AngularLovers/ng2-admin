#Taurus
纬度空间后台

## 一、项目启动方法
### 1. 设置镜像
`npm config set registry https://registry.npm.taobao.org `  淘宝的镜像,比国外的快n倍
### 2. 安装typescript编译工具
`npm install -g typings`
### 3.  安装依赖
`npm install`
#### 3.1. 如果发现node-sass安装失败 再执行一次 `npm install node-sass`，不报错就可以了。这源自于天朝的牛逼墙
### 4. 运行项目
`npm run start`
### 5. 访问项目
`http://localhost:3000`
